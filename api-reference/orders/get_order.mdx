---
title: "Get single order"
openapi: "GET /partner/v1/orders/{contractNumber}"
description: "Retrieve a single order by its number"
---

If you know the contract number of a particular order, you can use this endpoint to retrieve the order details.

This can be useful to follow the order processing and track status changes. See [order states](/guides/order_states) for an explanation about the different order states.
