---
title: 'List tariffs'
openapi: "GET /partner/v1/tariffs"
description: 'List the available tariffs'
---

This endpoint allows you to get a list of tariffs available for you to offer to customers. Use the `tariffKey` in subsequent calls to [calculate offer price](/api-reference/tariffs/calculate_price) and [create order](/api-reference/orders/create_order) calls
